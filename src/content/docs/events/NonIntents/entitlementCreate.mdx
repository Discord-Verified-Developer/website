<Aside title="Functions">

This event has a function to retrieve data:

[$newEntitlementData[option]](/functions/newentitlementdata)

[$oldEntitlementData[option]](/functions/oldentitlementdata)

</Aside>

<Example>
<div slot="default">

```js
client.entitlementCreateCommand({
    name: "Entitlement Create",
    channel: "$channelId",
    code: `
        $log[$newEntitlementData[userId] just purchased a new entitlement!]
    `
});
```

</div>
<div slot="handler">

```js
module.exports = {
    name: "Entitlement Create",
    channel: "$channelId",
    type: "entitlementCreate",
    code: `
        $log[$newEntitlementData[userId] just purchased a new entitlement!]
    `
};
```

</div>
