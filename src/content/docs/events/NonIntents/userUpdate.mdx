<Aside title="Functions">

This event has a function to retrieve data:

[$newUser[option]](/functions/newuser)

[$oldUser[option]](/functions/olduser)

</Aside>

<Example>

<div slot="default">

```js
client.userUpdateCommand({
    name: "User Update",
    channel: "$channelId",
    code: `
        $log[$oldUser[id]'s data changed!]
    `
});
```

</div>
<div slot="handler">

```js
module.exports = {
    name: "User Update",
    channel: "$channelId",
    type: "userUpdate",
    code: `
    $log[$oldUser[id]'s data changed!]
    `
};
```

</div>
</Example>
