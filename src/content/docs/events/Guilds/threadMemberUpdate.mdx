---
title: onThreadMemberUpdate
description: This event triggers whenever a member within a thread is updated.
id: threadMemberUpdate
---

import Example from "../../../../components/Page/Example.astro";
import { Aside } from '@astrojs/starlight/components';

<Aside title="Intents">

This requires the `Guilds` intent!

</Aside>

<Example>

<div slot="default">

```js
client.threadMemberUpdateCommand({
    name: "Thread Member Update",
    channel: "$channelId",
    code: `
        $log[$authorId just changed!]
    `
});
```

</div>
<div slot="handler">

```js
module.exports = {
    name: "Thread Member Update",
    channel: "$channelId",
    type: "threadMemberUpdate",
    code: `
        $log[$authorId just changed!]
    `
};
```

</div>
</Example>
